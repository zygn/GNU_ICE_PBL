# 소프트웨어 개발론 PBL 과제
- 경상대학교 정보통신공학과 D팀 (박성률, 신동준, 최윤도, 한동국)


## 개요
- 날씨에 따른 의류 매칭 시스템
- 이미지 분류, 날씨에 따른 가중치 연산으로 적절한 의류를 추천
- Cafe24 쇼핑몰에서 제공하는 이미지로 학습


## 구현

### [구현 완료] 날씨 URL, 의류 데이터를 입력하면 자동으로 적절한 의류를 추천 가능


- CNN (합성곱 신경망)

  - Python 3.7 or later
  - Tensorflow 2.0
  - Keras
  - Pillow
  
  
- 웹 크롤링

  - Beautifulsoup4 (4.8.0 or Later)
  - Urllib3
  
- XML 파싱

  - Urllib3
